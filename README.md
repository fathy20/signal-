# Signal Plus Website - Deployment & Notes

ملف يحتوي على النسخة الجاهزة لموقع "سيجنال بلس".

## المحتويات
- `index.html`  — الصفحة الرئيسية
- `styles.css`  — أنماط الموقع
- `script.js`   — جافاسكربت صغير للحركات والنموذج
- `assets/logo.svg` — شعار SVG بديل
- `contact.php` — مثال على معالجة النموذج باستخدام PHP
- `README.md`   — هذا الملف

## النشر على استضافة PHP (طريقة سريعة)
1. افتح حساب استضافة تدعم PHP (مثلاً أي استضافة مشتركة أو VPS مع Apache/Nginx + PHP).
2. انسخ كل ملفات المجلد إلى مجلد `public_html` أو `www` عبر FTP أو File Manager.
3. عدّل ملف `contact.php` وغيّر قيمة المتغير `$to` إلى بريدك الحقيقي.
4. تأكد أن الدالة `mail()` مفعّلة على الاستضافة. إن لم تعمل، استخدم مكتبة PHPMailer مع SMTP (مثلاً Gmail / Sendgrid).

### مثال سريع لإعداد PHPMailer
- ثبّت PHPMailer أو ضع ملفاته في المجلد ثم استخدم SMTP بدلاً من `mail()`:
```php
// استخدم PHPMailer وتعيين SMTP (مثال مختصر)
// require 'path/to/PHPMailer.php';
```

## تهيئة للشبكة وSEO
- غيّر وصف الميتا `description` وعناصر `title` حسب حملاتك.
- أضف صور منتجات داخل `assets/` وحدّث المسارات داخل `index.html`.

## جاهز للرفع عبر FTP
- ربط بيانات FTP من خلال ملف مدير الاستضافة أو أي برنامج مثل FileZilla.
- ارفع كل الملفات إلى المجلد العام.

## ملاحظات أمان
- هذا مثال تعليمي بسيط. قبل إطلاق الموقع على بيئة إنتاج:
  - اضف تحقق CSRF.
  - تحقق من صحة المدخلات أكثر.
  - سجّل الطلبات في قاعدة بيانات أو ملف لتتبعها.
  - استخدم اتصال HTTPS.

---

لو تحب أعمل لك:
- تركيب PHPMailer + مثال SMTP،
- إضافة صفحة لوحة تحكم صغيرة لعرض الطلبات (PHP + JSON file)،
- تصميم صور باقات جاهزة للـ `assets/`،
قلّي أي واحدة أبدأ بيها وسأعملها فوراً.
